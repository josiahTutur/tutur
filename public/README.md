# Static assets

Files here are served from the site root by Vite (e.g. `public/maya.png` → `/maya.png`).

## maya.png (Maya's portrait for the particle face)

Save the AI lady portrait here as **`maya.png`** (front-facing, plain/uniform
background — the background colour is auto-keyed out). The Intro page samples it
into the particle face. If the file is missing, the app falls back to a
procedural violet face automatically.
